# HTOP

![htop](screenshots/htop.png)

### UPTIME
Время непрерывной работы системы, берет информацию из /proc/uptime

### SWP
Swap используется, когда системе не хватает оперативной памяти

### TASKS
Tasks показывает количество процессов и потоков в системе
Процесс — это отдельная запущенная программа.
Поток — это часть процесса, которая выполняется внутри него

### Load Average
Средняя нагрузка на систему за 1, 5 и 15 минут
Если у сервера 1 CPU, то значение около 1.00 означает полную загрузку одного ядра

### PID
Идентификатор процесса

### USER
Пользователь, от которого был запущен процесс

### PRI
Приоритет процесса на уровне ядря ( NI -20 до 19, где увелечение < приоритет )

### S
Состояние процесса.
R - Running ( Выполняется ); S - Sleeping ( Ожидает события ); D - Disk sleep ( Ожидает диска/ввода-вывода )
Z - Zombie ( Зомби-процесс ); T - Stopped ( Остановлен )

## Полезные клавиши HTOP
F3 - Поиск процесса; F4 - Фильтр; F5 - Дерево; F6 - Сортировка; F9 - Завершение процесса; F10 - Выход

## WATCH
watch -n 1 ps -p - обновлять информацию об процессе каждую секунду
