# SSH

## SSH (Secure Shell, безопасная оболочка) — это сетевой протокол, предназначенный для безопасенного удаленного управления компьютерами и серверами через сеть.

## Клиент и сервер

SSH состоит из двух чайстей:
Client, который иницилизирует подключение через ssh user@192.168.1.1 (ssh -p 2222 user@192.168.1.1)
Server, на котором работает демон sshd. Он принимает входящие подключения и проверяет права доступа пользователей.
